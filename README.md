# java

This is an [openjdk](https://openjdk.org) implementation.  This container uses
the Alpine [package](https://pkgs.alpinelinux.org/packages?name=openjdk25*&branch=edge&repo=&arch=x86_64&origin=&flagged=&maintainer=)
to provide the Java environment.  This package by default uses the JRE to
build java code install the JDK in a container stage.
